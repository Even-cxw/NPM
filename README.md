### npm发包
- 包作者：evendetail

- 查看当前环境下的用户
```javascript
npm whoami
```

- 发包命令
```javascript
npm publish
```

- 查看包版本
```javascript
npm view vue-rate-compontents version
```

- 更新自己发版的npm包
```javascript
npm uninstall vue-rate-compontents
npm install vue-rate-compontents
```
